# HeatCast - Lennix Smith World Junior Championships

Live maneuver observation and analysis viewer. Watch the performance video with a real-time carousel that automatically scrolls to highlight each maneuver as it occurs.

## Features

- ⏯️ **Interactive Video Player** - Watch the full performance with standard video controls
- 🔄 **Real-time Carousel Sync** - Carousel automatically scrolls to show the active maneuver as it happens
- 🎯 **Click to Jump** - Click any maneuver card to jump the video to that moment
- 🎨 **Institutional Design** - Clean, minimal interface focused on observation
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile

## How to Use

1. **Press Play** - Use the video player controls to watch the performance
2. **Watch Carousel** - The maneuver carousel below automatically highlights the current maneuver
3. **Jump Ahead** - Click any maneuver card to jump to that moment in the video
4. **Scroll Cards** - Manually scroll the carousel left/right to preview all maneuvers
